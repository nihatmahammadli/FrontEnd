Özüm haqqında məlumat

Mən Nihat Məhəmmədli, Məncə Mən əyləncəli, çalışqan və diqqətli biriyəm.

  • Kitab oxumağı sevirəm
    • Əsasən fəlsəfi, klassikal və özünü inkişaf kitabları oxumağı sevirəm.
    • Ən sevdiyim kitab isə Marcus Aureliusdan Stoa fəlsəfəsi haqqında olan kitabı.
    • Ümumiyyətlə ən yaxşı dostlarım kitablar oldu.
    • Məncə hamı kitab oxumalıdı.
  • Film izləməyi sevirəm

    • (Sizə sevdiyim filmlərdən danışacam ümid edirəm məni tanımağa kömək edər)


    1. Ən sevdiyim 2 film Fight Club və Detachmentdir.
    2. Əsasən hərkətli və dərin filmləri sevirəm.
    3. Düşünürəm ki kitabllar qədər filmlərdə önəmlidir.
